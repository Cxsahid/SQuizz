# 🎯 SQuizz – Interactive Quiz Web App

## 📌 About

SQuizz is an interactive web-based quiz application designed to make learning fun and engaging.
It allows users to test their knowledge through dynamic quizzes while receiving instant feedback and scores.

Built using both **manual web development** and **AI-assisted techniques**, SQuizz focuses on simplicity, performance, and user experience.

---

## 🚀 Features

* 🎮 Interactive quiz gameplay
* ⏱️ Real-time scoring system
* 📊 Instant result display
* 💡 User-friendly interface
* 📱 Responsive design (works on mobile & desktop)

## ▶️ How to Run Locally

1. Clone the repository:

```

```

2. Open the folder:

```
cd SQuizz
```

3. Run the project:

* Open `index.html` in your browser
  OR
* Use Live Server in VS Code
